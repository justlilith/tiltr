# tiltr

[Live! https://tiltr.cc](https://tiltr.cc)

## Twitter app for browsing art!

Tiltr is an alternative to Twitter's normal UI. The focus is on art discovery and exploration.

## Frontend

- Elm

## Backend

- Node.js
- Twitter API

## Dependencies

- Elm
- NoRedInk/elm-json-decode-pipeline
- Node.js
- Express
- cors
- twitter
